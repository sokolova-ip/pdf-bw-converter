# PDF to Black & White Converter - Техническая документация

## Обзор проекта

**PDF to Black & White Converter** - это десктопное приложение с графическим интерфейсом для конвертации цветных PDF файлов в черно-белый формат с настройкой параметров изображения.

### Основные возможности:
- Конвертация PDF в черно-белый формат
- Предпросмотр с настройками в реальном времени
- Настройка яркости, контрастности, резкости
- Выбор размера выходного файла
- Автоматическое определение черно-белых PDF
- Сохранение ориентации страниц

## Архитектура проекта

### Структура проекта:

```text
pdf_bw_converter/
├── examples/               # Примеры изображений
├── src/                    # Исходный код
│   ├── __init__.py
│   ├── main.py            # Точка входа приложения
│   ├── converter.py       # Логика конвертации PDF
│   └── gui.py             # Графический интерфейс
├── tests/                  # Модульные тесты
│   ├── __init__.py
│   └── test_converter.py
├── docs/                   # Документация
│   └── README.md
├── requirements.txt        # Зависимости
├── setup.py               # Конфигурация пакета
└── run.py                 # Скрипт запуска
```
### Технологический стек:
- **Python 3.7+**
- **Tkinter** - графический интерфейс
- **PyMuPDF (fitz)** - работа с PDF
- **Pillow (PIL)** - обработка изображений
- **threading** - многопоточность для UI

## Установка и настройка

### Требования:
- Python 3.7 или выше
- 100 МБ свободного места
- Разрешения на чтение/запись файлов

### Установка:

1. **Клонирование репозитория:**
```bash
git clone https://github.com/sokolova-ip/pdf-bw-converter.git
cd pdf-bw-converter
```
2. **Установка зависимостей:**

```bash
pip install -r requirements.txt
```
3. **Запуск приложения:**

```bash
# Способ 1: Через run.py
python run.py

# Способ 2: Прямой запуск
python src/main.py
```
### Зависимости:
PyMuPDF==1.23.8

Pillow==10.0.1

# Использование

## Базовые шаги:
1. **Запустите** `python run.py`
2. **Выберите PDF** через кнопку "Обзор"
3. **Настройте параметры** с помощью ползунков
4. **Просмотрите** результат в окне предпросмотра
5. **Конвертируйте** и сохраните файл

## Настройки изображения:

- **Яркость:** 0.1 - 3.0
- **Контрастность:** 0.1 - 3.0  
- **Резкость:** 0.1 - 3.0
- **Качество JPEG:** 10 - 100

## Размеры вывода:

- original (оригинальный размер)
- A4
- A3
- A5
- Letter
- Legal

**Сохранение ориентации** - опция для сохранения оригинальной ориентации страниц

# Устранение неполадок

## Приложение не запускается:
- Проверить Python 3.7+
- Проверить зависимости
- Проверить права доступа

## PDF не загружается:
- Проверить целостность файла
- Проверить права доступа
- Убедиться, что не защищен паролем

## Плохое качество:
- Увеличить качество JPEG
- Настроить яркость/контрастность
